Protean-shrx
============

> Rx utility methods for shell-like interaction


Summary
-------


Installation
------------

~~~
% npm install protean-shrx
~~~

Dependencies
------------

These are installed when **protean-shrx** is installed.

~~~
protean:    0.2.x
underscore: 1.x
del:        1.x
globby:     1.x
~~~


### Development Dependencies ###

Installed when you run `npm link` in the package directory.

~~~
mocha:    2.x.x
should:   4.x.x
gulp:     3.x.x
gulp-ejs: 0.x.x
~~~
